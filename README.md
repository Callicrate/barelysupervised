# BarelySupervised Codex plugin marketplace

This repository is a Codex plugin marketplace generated by `bs plugin`. It exposes the BarelySupervised roster as an installable plugin.

## Deploy

1. Push this directory to a GitHub repository.
2. Add it as a marketplace source:

   ```
   codex plugin marketplace add <owner>/<repo>
   ```

3. In the ChatGPT desktop app, open Plugins, select the BarelySupervised marketplace, and install the plugin.
4. Trust the SessionStart hook with `/hooks` so it can run.

## Layout

- `.agents/plugins/marketplace.json` is the marketplace catalog.
- `plugins/barelysupervised/` is the plugin itself.

## What the plugin does

On session start the plugin injects the main-agent routing context and, unless `BARELYSUPERVISED_SKIP_AGENT_INSTALL` is set, installs the bundled subagent profiles into the project's `.codex/agents` directory without overwriting existing files.

Regenerate this repository with `bs plugin` after changing the active roster.
